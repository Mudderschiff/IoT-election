653/***************************** Certificate Macros *****************************
654 *
655 * The section below defines macros used in typically all of the wolfSSL
656 * examples such as the client and server for certs stored in header files.
657 *
658 * There are various certificate examples in this header file:
659 * https://github.com/wolfSSL/wolfssl/blob/master/wolfssl/certs_test.h
660 *
661 * To use the sets of macros below, define *one* of these:
662 *
663 * USE_CERT_BUFFERS_1024 - ECC 1024 bit encoded ASN1
664 * USE_CERT_BUFFERS_2048 - RSA 2048 bit encoded ASN1
665 * WOLFSSL_SM[2,3,4] - SM Ciphers
666 *
667 * For example: define USE_CERT_BUFFERS_2048 to use CA Certs used in this
668 * wolfSSL function for the `ca_cert_der_2048` buffer, size and types:
669 *
670 * ret = wolfSSL_CTX_load_verify_buffer(ctx,
671 * CTX_CA_CERT,
672 * CTX_CA_CERT_SIZE,
673 * CTX_CA_CERT_TYPE);
674 *
675 * See https://www.wolfssl.com/documentation/manuals/wolfssl/group__CertsKeys.html#function-wolfssl_ctx_load_verify_buffer
676 *
677 * In this case the CTX_CA_CERT will be defined as `ca_cert_der_2048` as
678 * defined here: https://github.com/wolfSSL/wolfssl/blob/master/wolfssl/certs_test.h
679 *
680 * The CTX_CA_CERT_SIZE and CTX_CA_CERT_TYPE are similarly used to reference
681 * array size and cert type respectively.
682 *
683 * Similarly for loading the private client key:
684 *
685 * ret = wolfSSL_CTX_use_PrivateKey_buffer(ctx,
686 * CTX_CLIENT_KEY,
687 * CTX_CLIENT_KEY_SIZE,
688 * CTX_CLIENT_KEY_TYPE);
689 *
690 * see https://www.wolfssl.com/documentation/manuals/wolfssl/group__CertsKeys.html#function-wolfssl_ctx_use_privatekey_buffer
691 *
692 * Similarly, the other macros are for server certificates and keys:
693 * `CTX_SERVER_CERT` and `CTX_SERVER_KEY` are available.
694 *
695 * The certificate and key names are typically `static const unsigned char`
696 * arrays. The [NAME]_size are typically `sizeof([array name])`, and the types
697 * are the known wolfSSL encoding type integers (e.g. WOLFSSL_FILETYPE_PEM).
698 *
699 * See `SSL_FILETYPE_[name]` in
700 * https://github.com/wolfSSL/wolfssl/blob/master/wolfssl/ssl.h
701 *
702 * See Abstract Syntax Notation One (ASN.1) in:
703 * https://github.com/wolfSSL/wolfssl/blob/master/wolfssl/wolfcrypt/asn.h
704 *
705 * Optional SM4 Ciphers:
706 *
707 * Although the SM ciphers are shown here, the `certs_test_sm.h` may not yet
708 * be available. See:
709 * https://github.com/wolfSSL/wolfssl/pull/6825
710 * https://github.com/wolfSSL/wolfsm
711 *
712 * Uncomment these 3 macros to enable the SM Ciphers and use the macros below.
713 */
